# poisonplants
Java Swing poison plant checker using trefle's global plants API

Trefle's API is a little touchy. Many entries to the API have largely null values. Examples of entries that have complete data in the API:

"southern red oak"
"little green sedge",
"tietongue",
"hairy lespedeza",
"Wright's deervetch",
"rough bluegrass",
"desert bitterbrush",
"littletree willow",
"gulf cordgrass",
"large yellow vetch",
"common blue violet",
"mescal bean",
"tiplant",
"seagrape",
"nineleaf biscuitroot",
"marsh muhly"
